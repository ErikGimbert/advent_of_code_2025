## Init python env

* **In VSCode**: `F1` > `Task: Run Build Task`
* **With uv**: `uv sync`

## How to launch ?

### In VSCode

* `Ctrl+F5`: launch the program for the last day
* `F1` > `Task: Run Test Task`: launch all tests

### In shell (with )

* `python src/main.py`: launch the program for the last day
* `python src/main.py X`: launch the program for the day X
* `python src/main.py -a`: launch the program for all the days
* `python src/main.py -h`: for help message
